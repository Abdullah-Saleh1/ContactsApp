## ✨ Features

- **Modern Dark UI:** Fully customized controls (DataGrid, ComboBox, ContextMenu) with a consistent dark theme and smooth button animations.
- **3-Tier Architecture:** Built with clean separation of concerns:
  - `PresentationLayer` (WPF UI)
  - `BusinessLayer` (Business Logic)
  - `DataAccessLayer` (Database Communication)
- **Smart Search:** Instant real-time filtering in the DataGrid with a clear search utility.
- **Full CRUD Support:** Add, Edit, Delete, and View contact profiles smoothly.
- **Profile Image Integration:** Easily upload or remove contact profile pictures.

---

## 🚀 Tech Stack & Tools

- **Language:** C#
- **Framework:** WPF (.NET Framework)
- **Architecture:** 3-Tier Architecture
- **UI Styling:** Custom XAML Styles & ControlTemplates
- **Database:** SQL Server / ADO.NET
